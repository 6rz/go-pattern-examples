# 发布订阅模式

发布-订阅是一种消息传递模式，基本设计原则是将消息发布者和提供者分开，常常用于在不同组件之间传递消息。

![图示关系](../../images/pub-sub-pattern-0.png)
发布订阅模式往往实现为<消息代理，消息中间件，消息队列>等

该模式中的三个关键类型：消息本身、消息主题、订阅用户。
